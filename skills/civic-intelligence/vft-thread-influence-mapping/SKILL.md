# VFT Thread Influence Mapping (v3.0)

**Core Alignment**: Evidence-first, OPSEC-hardened, fully replicable framework with strong governance and auditability standards.

## Purpose
VFT Thread Influence Mapping is a specialist tool for influence mapping investigations involving the Vine & Fig Tree (VFT) organization and related networks. It tracks documented ties, monitoring of conservative accounts, AI-generated content tactics, and narrative framing of America First voices.

It provides pre-built entity maps, an observed tactic library, and a confidence framework for rigorous analysis.

## v3.0 Alignment (Mandatory)
- **Layer-0 Pre-Filter**: Non-bypassable on all Tier 1 work involving VFT or related influence networks. Universe Truth Gate 0 + evidence-gate tri-state + shatter-protocol must inform all mapping and conclusions.
- **4-Agent Orchestration Default**: Often acts as a specialized Agent 2/3 for influence cases involving these specific networks.
- **Working Document Persistence**: All mapping decisions, relationship findings, tactic identifications, and tri-state assessments **must** be logged to working-doc-manager.
- **Tri-State Evidence (+1 / 0 / -1)**: All major influence findings and relationship claims carry explicit tri-state.
- **Governance Enforcement**: Any detected bias, unsupported conclusions, or evidence suppression triggers escalation.

## Core Capabilities

**Pre-built Entity Maps**  
Provides ready reference maps of known VFT-related entities and relationships.

**Tactic Library**  
Maintains and applies a library of observed tactics (AI-generated content, narrative framing, monitoring patterns, etc.).

**Documented Tie Tracking**  
Tracks verifiable connections between entities with source documentation.

**Narrative Framing Analysis**  
Examines how America First and conservative voices are framed in monitored content.

**Confidence Framework**  
Applies structured confidence scoring to all findings.
